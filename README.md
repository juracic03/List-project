# Opis i specifikacija

Mali projekt o listi za kupnju proizvoda. Projektić sadržava input polje za ispunjenje podataka i dodavanje u listu. Dodana stavka se može označiti kao riješena (kupljena) ili se obrisati. Na dnu je prikaz podataka o proizvodima na listi i kupnja istih.

# Alati

Upotreba JavaScript-a, HTML-a i CSS-a za dodavanje i brisanje sadržaja te vizualni dizajn  liste.
